# Demo Video Creator (Frontend)

## Setup

- Install dependencies:

```
cd frontend
npm install
```

- Configure backend base URL (create a `.env` file in this folder):

```
VITE_BACKEND_BASE=http://localhost:8000
```

## Run

```
npm run dev
```

Open the printed local URL (usually `http://localhost:5173`).

## Build

```
npm run build
npm run preview
```
